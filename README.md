# 🌟 Skynote - Révise plus vite, Réussis mieux

Une application web SaaS éducative qui transforme tes cours en fiches de révision intelligentes grâce à l'IA.

## ✨ Fonctionnalités

- 📤 **Upload intelligent** : PDF, photos, audio, texte
- 🤖 **IA puissante** : Génération automatique de fiches
- ❓ **QCM personnalisés** : Questions adaptées à ton niveau  
- 💰 **Sky Coins** : Système de gamification et récompenses
- 🌓 **Dark/Light mode** : Pour réviser jour et nuit
- 📱 **Responsive** : Fonctionne sur tous les appareils
